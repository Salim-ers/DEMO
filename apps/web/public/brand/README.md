# Logo Studio One

Déposez ici les exports officiels du logo — ils s'affichent automatiquement
partout dans l'app (hero, header, sidebar, footer, CTA, brand kit), à la bonne
taille, sans aucune modification de code.

| Fichier                    | Usage                                              |
| -------------------------- | -------------------------------------------------- |
| `studio-one.png`           | Badge circulaire bronze (logo principal)           |
| `studio-one-white.png`     | Lettrage blanc sur fond transparent (zones sombres)|

- Format conseillé : PNG carré à fond transparent, ≥ 600×600 px.
- Tant que ces fichiers sont absents, un rendu vectoriel de secours fidèle est
  affiché (jamais d'image cassée).
